## 4.2隐式序列

惰性计算。

迭代器是一种对象，提供对值逐一顺序访问的功能。

```py
>>> iterator = iter(primes)
>>> next(iterator)
>>> try:
        next(iterator)
    except StopIteration:#没有更多可用值
        print('No more values')
```

Python 中的 `for` 语句是对迭代器进行操作。 如果对象具有返回迭代器的 `__iter__` 方法（method），则表示对象是可迭代的。
